Git 101
====
* `git clone <url>` = copy repository from `url`
* `git checkout -b <new-branch-name>` = new branch
* `git add -A` = Mark everything for commit
* `git commit`
* `git pull --rebase upstream master`
* `git push origin <branch-name>`
